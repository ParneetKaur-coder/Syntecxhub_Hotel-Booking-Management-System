# Hotel Booking Management System

A simple Java Command Line Interface (CLI) application for managing hotel room bookings. This project is developed using 
Object-Oriented Programming (OOP) concepts and demonstrates room booking, cancellation, room availability management, and
input validation.

## Features

* View Available Rooms
* Book a Room
* Cancel Booking
* View All Rooms
* Input Validation
* Exception Handling

## Technologies Used

* Java
* ArrayList
* Object-Oriented Programming (OOP)
